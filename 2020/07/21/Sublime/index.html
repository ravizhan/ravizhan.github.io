## 设置中文/汉化

安装插件控制面板Tools -> Install Package Control
安装有点慢，在左下角有提示

点击Preferences -> Package Control

在弹出的窗口中点击Package Control: Install Package

再在弹出的窗口中，输入clz，点击ChineseLocalizations
